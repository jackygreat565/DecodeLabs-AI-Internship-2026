## ⚙️ Setting Up Your Environment

Before you run the software, you must install the Python language. 

1. Go to the official Python website.
2. Select the latest version of Python for Windows.
3. Download the installer and run it.
4. IMPORTANT: Check the box labeled "Add Python to PATH" before you click "Install Now." This step is necessary for your computer to find the files.

After you install Python, press the Windows key on your keyboard, type "cmd," and press Enter. This opens the command prompt. Type `python --version` and press Enter. If you see a version number, your computer is ready.

## 📦 Installing Project Dependencies

Each project requires extra libraries to function correctly. These libraries let the code perform tasks like image processing and data analysis.

1. Open the folder where you extracted the project files.
2. Hold the Shift key and right-click on an empty space inside the folder.
3. Select "Open PowerShell window here" or "Open in Terminal."
4. Type the command `pip install -r requirements.txt` and press Enter.
5. Wait for the process to finish. It will automatically download the necessary tools for the chatbot, classification, recommendation, and vision projects.

## 🔮 Exploring the Projects

This workbench contains four distinct AI modules. 

### Rule-Based Chatbot
This project uses logic and patterns to hold a conversation. You provide input, and the bot matches keywords to predefined answers. It demonstrates how developers build bots without requiring complex large language models.

### Supervised Classification
This tool categorizes data points based on examples it has seen before. Use this to sort text or numerical data into groups. It relies on internal logic to predict the category of new information.

### Content-Based Recommendation System
This system suggests items based on their properties. If you like a specific type of movie or product, the system calculates similarities and identifies other items with matching traits.

### Computer Vision (OCR and Object Detection)
These scripts use OpenCV and Tesseract to interpret images. The OCR tool extracts text from photographs of documents. The object detection tool identifies specific items within an image, such as cars or people, and draws boxes around them.

## 🔍 Troubleshooting Common Issues

If the software does not start, check for the following common issues:

*   **Missing Libraries:** If you see an error about a missing library, run the installation command in the "Installing Project Dependencies" section again.
*   **Path Errors:** Ensure you checked "Add Python to PATH" during installation. If you missed this, reinstall Python and check the box.
*   **Wrong Folder:** Ensure your terminal window is inside the specific folder containing the `main.py` file for that project.
*   **Terminal Permissions:** Close your terminal and open it again as an administrator if you encounter problems with file access.
